CSS
CSS is used for formatting of the webpage.CSS is the language we use to style an html Document.CSS describes how html elements should be displayed.IN CSS is used many formatting like color,background color,text align,bold,italic,underline, border etc..
Javascript
Javascript is a lightweight,interpreted programming language.It is designed for creating network-centric applications. It is complimentary to and integrated with java.Javascript is very easy to implement because it is intergrated with HTML. It is open and close-platform.
